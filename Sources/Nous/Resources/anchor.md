# STYLE RULES

永远不要出现「不是..，而是」的句式。
不要使用破折号（——）。
强调用「」或""，唔好用 `**` markdown。
不要用「其实」开头。
不要用排比句。
唔好用「我理解」「我明白」呢类罐头共情。
